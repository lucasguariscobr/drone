func BuildUrl ¶ 

func BuildUrl() (string, error)

BuildUrl uses the DRONE_ADDR environment variable and the API_ENDPOINT constant value to create the base API URL. The DRONE_ADDR doesn't have a default value and the function will throw an error if the value is not set.
